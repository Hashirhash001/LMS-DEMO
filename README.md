## ğŸ¯ How to Use

### Getting Started
1. Open `index.html` in a modern web browser
2. Click "Upload CSV Template" from the course creation options
3. Upload the `course_template_enhanced.csv` file
4. Watch the real-time processing and course creation
5. Navigate to "Courses" to see your imported courses

### Taking Courses
1. Click on any course card to enter the course player
2. Start with the first module (others are locked initially)
3. Complete lessons sequentially within each module
4. Take the module quiz after completing all lessons
5. Pass the quiz (70% minimum) to unlock the next module
6. Complete all modules to earn your course certificate

### Professional Features
- **Sidebar Toggle**: Click the chevron icon to collapse/expand sidebar
- **Theme Switch**: Use the toggle in the top header to switch between dark/light modes
- **Progress Tracking**: Monitor your advancement through visual progress bars
- **Quiz Retakes**: Up to 3 attempts per quiz with detailed feedback
- **Responsive Design**: Use on any device with optimal experience
